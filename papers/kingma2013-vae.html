Abstract

How can we perform efficient inferenceterm·key and learning in directed probabilistic models, in the presence of continuous latentterm·key variables with intractable posteriorterm·key distributions, and large datasets? We introduce a stochastic variational inferenceterm·mth and learning algorithm that scales to large datasets and, under some mild differ- entiability conditions, even works in the intractable case. Our contributions are two-fold. First, we show that a reparameterizationterm·key of the variationalterm·key lower bound yields a lower bound estimatorterm·key that can be straightforwardly optimized using stan- dard stochastic gradientterm·key methods. Second, we show that for i.i.d. datasets with continuous latentterm·key variables per datapoint, posteriorterm·key inferenceterm·key can be made espe- cially efficient by fitting an approximate inferenceterm·key model (also called a recogni- tion model) to the intractable posteriorterm·key using the proposed lower bound estimatorterm·key. Theoretical advantages are reflected in experimental results.

Introduction

How can we perform efficient approximate inferenceterm·key and learning with directed probabilistic models whose continuous latentterm·key variables and/or parameters have intractable posteriorterm·key distributions? The variationalterm·key Bayesianterm·stat (VB) approach involves the optimization of an approximation to the intractable posteriorterm·key. Unfortunately, the common mean-field approach requires analytical solutions of expecta- tions w.r.t. the approximate posteriorterm·key, which are also intractable in the general case. We show how a reparameterizationterm·key of the variationalterm·key lower bound yields a simple differentiable unbiased estimatorterm·key of the lower bound; this SGVB (Stochastic Gradient Variational Bayes) estimatorterm·def can be used for ef- ficient approximate posteriorterm·key inferenceterm·key in almost any model with continuous latentterm·key variables and/or parameters, and is straightforward to optimize using standard stochastic gradientterm·key ascent techniques.

For the case of an i.i.d. dataset and continuous latentterm·key variables per datapoint, we propose the Auto- Encoding VB (AEVB) algorithm. In the AEVB algorithm we make inferenceterm·key and learning especially efficient by using the SGVB estimatorterm·key to optimize a recognition modelterm·def that allows us to perform very efficient approximate posteriorterm·key inferenceterm·key using simple ancestral sampling, which in turn allows us to efficiently learn the model parameters, without the need of expensive iterative inferenceterm·key schemes (such as MCMC) per datapoint. The learned approximate posteriorterm·key inferenceterm·key model can also be used for a host of tasks such as recognition, denoising, representation and visualization purposes. When a neural network is used for the recognition modelterm·def, we arrive at the variationalterm·key auto-encoder.

2.1 Problem scenario

Let us consider some dataset X = {x(i)}N i=1 consisting of N i.i.d. samples of some continuous or discrete variable x. We assume that the data are generated by some random process, involving an unobserved continuous random variable z. The process consists of two steps: (1) a value z(i) is generated from some prior distribution pθ∗ (z); (2) a value x(i) is generated from some condi- tional distribution pθ∗ (x|z). We assume that the prior pθ∗ (z) and likelihoodterm·key pθ∗ (x|z) come from parametric families of distributions pθ(z) and pθ(x|z), and that their PDFs are differentiable almost everywhere w.r.t. both θ and z. Unfortunately, a lot of this process is hidden from our view: the true parameters θ∗ as well as the values of the latentterm·key variables z(i) are unknown to us.

Very importantly, we do not make the common simplifying assumptions about the marginal or pos- terior probabilities. Conversely, we are here interested in a general algorithm that even works effi- ciently in the case of:

1. Intractability:

the case where the integral of the marginal

likelihoodterm·key pθ(x) = (cid:82) pθ(z)pθ(x|z) dz is intractable (so we cannot evaluate or differentiate the marginal like- lihood), where the true posteriorterm·key density pθ(z|x) = pθ(x|z)pθ(z)/pθ(x) is intractable (so the EM algorithm cannot be used), and where the required integrals for any reason- able mean-field VB algorithm are also intractable. These intractabilities are quite common and appear in cases of moderately complicated likelihoodterm·key functions pθ(x|z), e.g. a neural network with a nonlinear hidden layer.

2. A large dataset: we have so much data that batch optimization is too costly; we would like to make parameter updates using small minibatches or even single datapoints. Sampling- based solutions, e.g. Monte Carlo EM, would in general be too slow, since it involves a typically expensive sampling loop per datapoint.

We are interested in, and propose a solution to, three related problems in the above scenario:

1. Efficient approximate ML or MAP estimation for the parameters θ. The parameters can be of interest themselves, e.g. if we are analyzing some natural process. They also allow us to mimic the hidden random process and generate artificial data that resembles the real data.

2. Efficient approximate posteriorterm·key inferenceterm·key of the latent variableterm·dom z given an observed value x

for a choice of parameters θ. This is useful for coding or data representation tasks.

3. Efficient approximate marginal inferenceterm·key of the variable x. This allows us to perform all kinds of inferenceterm·key tasks where a prior over x is required. Common applications in computer vision include image denoising, inpainting and super-resolution.

For the purpose of solving the above problems, let us introduce a recognition modelterm·def qφ(z|x): an approximation to the intractable true posteriorterm·key pθ(z|x). Note that in contrast with the approximate posteriorterm·key in mean-field variational inferenceterm·mth, it is not necessarily factorial and its parameters φ are not computed from some closed-form expectation. Instead, we’ll introduce a method for learning the recognition modelterm·def parameters φ jointly with the generative modelterm·ai parameters θ.

From a coding theory perspective, the unobserved variables z have an interpretation as a latentterm·key representation or code. In this paper we will therefore also refer to the recognition modelterm·def qφ(z|x) as a probabilistic encoder, since given a datapoint x it produces a distribution (e.g. a Gaussian) over the possible values of the code z from which the datapoint x could have been generated. In a similar vein we will refer to pθ(x|z) as a probabilistic decoder, since given a code z it produces a distribution over the possible corresponding values of x.

4 Related work

The wake-sleep algorithm [HDFN95] is, to the best of our knowledge, the only other on-line learn- ing method in the literature that is applicable to the same general class of continuous latent variableterm·dom models. Like our method, the wake-sleep algorithm employs a recognition modelterm·def that approximates the true posteriorterm·key. A drawback of the wake-sleep algorithm is that it requires a concurrent optimiza- tion of two objective functions, which together do not correspond to optimization of (a bound of) the marginal likelihoodterm·key. An advantage of wake-sleep is that it also applies to models with discrete latentterm·key variables. Wake-Sleep has the same computational complexity as AEVB per datapoint.

Stochastic variational inferenceterm·mth [HBWP13] has recently received increasing interest. Recently, [BJP12] introduced a control variate schemes to reduce the high variance of the na ̈ıve gradientterm·key estimatorterm·key discussed in section 2.1, and applied to exponential family approximations of the poste- rior. In [RGB13] some general methods, i.e. a control variate scheme, were introduced for reducing the variance of the original gradientterm·key estimatorterm·key. In [SK13], a similar reparameterizationterm·key as in this paper was used in an efficient version of a stochastic variational inferenceterm·mth algorithm for learning the natural parameters of exponential-family approximating distributions.

The AEVB algorithm exposes a connection between directed probabilistic models (trained with a variationalterm·key objective) and auto-encoders. A connection between linear auto-encoders and a certain class of generative linear-Gaussian models has long been known. In [Row98] it was shown that PCA corresponds to the maximum-likelihoodterm·key (ML) solution of a special case of the linear-Gaussian model with a prior p(z) = N (0, I) and a conditional distribution p(x|z) = N (x; Wz, (cid:15)I), specifically the case with infinitesimally small (cid:15). In relevant recent work on autoencoders [VLL+10] it was shown that the training criterion of un- regularized autoencoders corresponds to maximization of a lower bound (see the infomax princi- ple [Lin89]) of the mutual information between input X and latentterm·key representation Z. Maximiz- ing (w.r.t. parameters) of the mutual information is equivalent to maximizing the conditional en- tropy, which is lower bounded by the expected loglikelihood of the data under the autoencoding model [VLL+10], i.e. the negative reconstrution error. However, it is well known that this recon- struction criterion is in itself not sufficient for learning useful representations [BCV13]. Regular- ization techniques have been proposed to make autoencoders learn useful representations, such as denoising, contractive and sparse autoencoder variants [BCV13]. The SGVB objective contains a regularization term dictated by the variationalterm·key bound (e.g. eq. (10)), lacking the usual nuisance regu- larization hyperparameter required to learn useful representations. Related are also encoder-decoder architectures such as the predictive sparse decomposition (PSD) [KRL08], from which we drew some inspiration. Also relevant are the recently introduced Generative Stochastic Networks [BTL13] where noisy auto-encoders learn the transition operator of a Markov chain that samples from the data distribution. In [SL10] a recognition modelterm·def was employed for efficient learning with Deep Boltz- mann Machines. These methods are targeted at either unnormalized models (i.e. undirected models like Boltzmann machines) or limited to sparse coding models, in contrast to our proposed algorithm for learning a general class of directed probabilistic models.

The recently proposed DARN method [GMW13], also learns a directed probabilistic model using an auto-encoding structure, however their method applies to binary latentterm·key variables. Even more recently, [RMW14] also make the connection between auto-encoders, directed proabilistic models and stochastic variational inferenceterm·mth using the reparameterizationterm·key trick we describe in this paper. Their work was developed independently of ours and provides an additional perspective on AEVB.

6 Conclusion

We have introduced a novel estimatorterm·key of the variationalterm·key lower bound, Stochastic Gradientterm·key VB (SGVB), for efficient approximate inferenceterm·key with continuous latentterm·key variables. The proposed estima- tor can be straightforwardly differentiated and optimized using standard stochastic gradientterm·key meth- ods. For the case of i.i.d. datasets and continuous latentterm·key variables per datapoint we introduce an efficient algorithm for efficient inferenceterm·key and learning, Auto-Encoding VB (AEVB), that learns an approximate inferenceterm·key model using the SGVB estimatorterm·key. The theoretical advantages are reflected in experimental results.

7 Future work

Since the SGVB estimatorterm·key and the AEVB algorithm can be applied to almost any inferenceterm·key and learning problem with continuous latentterm·key variables, there are plenty of future directions: (i) learning hierarchical generative architectures with deep neural networks (e.g. convolutional networks) used for the encoders and decoders, trained jointly with AEVB; (ii) time-series models (i.e. dynamic Bayesianterm·stat networks); (iii) application of SGVB to the global parameters; (iv) supervised models with latentterm·key variables, useful for learning complicated noise distributions.
