content-observations: finding · quantity · variable · definition · key-term — grounded in the full text
findingFrom high-frequency data, log-volatility behaves like a fractional Brownian motion with Hurst exponent H≈0.1 (H<1/2, rough) at all reasonable time scales — motivating the Rough FSV (RFSV) model, which fits financial time series and improves realized-volatility forecasts. — “Our main result is that log-volatility behaves essentially as a fractional Brownian motion with Hurst exponent H of order 0.1, at any reasonable time scale... We call our model Rough FSV (RFSV) to underline that, in contrast to FSV, H < 1/2.”
findingAlthough volatility is NOT long-memory under RFSV, classical persistence tests spuriously detect long memory in RFSV-generated data — explaining why long memory became an accepted stylized fact. — “classical statistical procedures aiming at detecting volatility persistence tend to conclude the presence of long memory in data generated from it. This sheds light on why long memory of volatility has been widely accepted as a stylized fact.”
